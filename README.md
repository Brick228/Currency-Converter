## Описание программы

**Currency Converter** - это GUI-приложение, которое позволяет конвертировать разные валюты актуально при помощи внешнего API и сохранять историю в .JSON формате

### Возможности:
- Конвертации из одной валюты в другую
- Сохранение истории в формате .JSON на своём компьютере

## Инструкция по запуску

### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter (обычно идёт в комплекте с Python)

### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
   ```bash
   cd bookTracker
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```

## Примеры использования

### Конвертирование:
1. Заполните все поля формы:
   - **Выберите 1 валюту:** RUB
   - **Выберите 2 валюту:** USD
   - **Выберите количество валюты:** 1000
2. Нажмите кнопку **"Конвертировать"**
3. Немного ниже появится ответ который также отобразится в истории
